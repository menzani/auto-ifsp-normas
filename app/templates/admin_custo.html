Como usar este painel
  1. Confirme os preços do modelo. Acesse o site da AWS sobre Bedrock, localize o modelo {{ model_id }} e insira os valores de entrada e saída acima. Clique em Confirmar preços para registrar que os valores estão atualizados. Repita sempre que a AWS anunciar reajuste.
  2. Acompanhe o custo por ano. Cada ano exibe o total faturado pela AWS (via Cost Explorer) e o valor convertido para reais. Abra o ano para ver o detalhamento por mês e por tipo de operação (extração, FAQ, revogação).
  3. Entenda a estimativa. A estimativa é calculada internamente com base nos tokens registrados no log multiplicados pelo preço configurado. Quando os preços estão corretos, a diferença entre estimativa e cobrado pela AWS indica créditos ou descontos da AWS.
  4. A cotação do dólar é automática. Atualizada a cada hora via open.er-api.com. Em caso de falha, usa o último valor registrado. O valor pode ser ajustado manualmente se necessário.

Como os valores são calculados

Cada operação registra os tokens consumidos no log de auditoria. O custo é calculado como: tokens × preço por 1M tokens, separando entrada e saída.

  • Extração — Claude Vision lê as páginas do PDF e gera o texto estruturado
  • FAQ — Claude gera perguntas frequentes a partir do texto extraído
  • Revogação — Claude gera o resumo de normativos revogados

Extração e FAQ ocorrem juntos a cada upload. A fórmula detalhada aparece ao expandir cada mês na tabela.
